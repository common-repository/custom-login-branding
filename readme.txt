Customize the wordpress login branding

== Description ==

A quick process for changing the branding for the wp-admin login screen.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `the entire contents of the zip file to your plugin directory '/wp-content/plugins/'
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a new branding image. Save it as 'logo-login.gif'
4. Upload the new login background image to the /wp-content/plugins/custom-login-branding/
